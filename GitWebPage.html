GitHubで作成したリポジトリをwebページとして公開する手順



<前提>

・GitHubのアカウント作成済み


<環境>

・Mac OS X Yosemite(ver.10.11.1)


<手順>

１.　GitHubで新しいリポジトリを作成

リポジトリ名、リポジトリの説明、その他設定をする。

2.　作成したリポジトリを開き、Settingsを選択する。

3.　GitHub Pages の項目の Source のオプションで ‘master branch’を選択し’Save’する。

4.　GitHub Pages の上の方に’ Your site is published at https://ユーザ名.github.io/リポジトリ名/’のように表示されるのでそのURLにアクセスするとWebページが表示される。
